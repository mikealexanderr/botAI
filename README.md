# botAI
Bot ini bisa mengklasifikasikan kupu-kupu dan ngengat!

## Requirements and How to Do
### Requirements
* Tensorflow 2.13
* Pillow
* Numpy
* Request
* Discord.py
### How to do
1. Run file Python
2. Upload an image to discord
3. Type the command $check
4. Wait to see the result of the classification
